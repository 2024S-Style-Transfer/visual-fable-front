---

## Usage

1. Git Clone

```cmd
$ git clone https://github.com/2024S-Style-Transfer/visual-fable-front.git
```

2. Install package (\* yarn을 추천드립니다.)

```cmd
$ yarn

# or

$ npm install
```

3. Start

```cmd
$ yarn dev

# or

$ npm run dev
```

4. Build & Deploy (TBD)

```cmd
$ yarn build

# or

$ npm run build
```
